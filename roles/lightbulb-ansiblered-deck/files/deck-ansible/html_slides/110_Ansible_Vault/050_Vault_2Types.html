ANSIBLE VAULT

Encrypt an entire file or just a string inside a non-encrypted file.

Ansible Terminal : Encrypted File
 cat ronin_secret.yml 

$ANSIBLE_VAULT;1.1;AES256

306637626335643365630a653066396533313738316266653

fonzie_show: Happy Days

We can encrypt an entire file or just a string inside an encrypted file.

We'll call the FULL FILE ENCRYPTION in this example, "Ronin".

We'll call the ENCRYPTED STRING in a plain-text file, "Fonzie"
